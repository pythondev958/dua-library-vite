# Starter Library Files

Put your initial app files in these folders, then run:

```bash
npm run starter:manifest
```

Folders:

- `duas/`
- `surahs/`
- `quran/`
- `morning/`
- `evening/`
- `other/`

Supported files:

- `.png`
- `.jpg`
- `.jpeg`
- `.webp`
- `.gif`
- `.pdf`

These starter files are included in the deployed app. User uploads still stay private in the browser.
